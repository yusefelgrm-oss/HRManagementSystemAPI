# HR Management System API

A simple RESTful API built with **ASP.NET Core 8** for employee management and user authentication using JWT.

## Features
- Employee CRUD Operations
- User Registration & Login
- JWT Authentication
- Entity Framework Core
- SQL Server
- Swagger API Documentation

## Technologies
- ASP.NET Core 8
- C#
- Entity Framework Core
- SQL Server
- JWT
- Swagger
